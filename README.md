# idea-garden

A low-friction inbox for rough ideas that don't belong to any single repo yet.

When a thought, question, or "someday" project can't be tied to an existing
repo, it lands here as an issue. Capture it loosely, then add comments, links,
and cross-references over time as it grows (or let it sit).

## How to use it

- **One idea per issue.** Title it loosely; the body can be a single sentence.
- **Label as you go.** Suggested labels: `idea`, `research`, `question`,
  `someday`, `reference`.
- **Link liberally.** Reference other issues with `couimet/idea-garden#NN`, or
  point at any other repo with `owner/repo#NN`.
- **Promote when ready.** If an idea grows into real work, open a repo (or move
  it to an existing one) and link back here.

## Private mirror

Ideas that can't be public live in
[`idea-garden-private`](https://github.com/couimet/idea-garden-private). The two
cross-reference each other; default to the private repo when in doubt, since
public is effectively permanent.
